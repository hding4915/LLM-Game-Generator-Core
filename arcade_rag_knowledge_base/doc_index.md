# The Python Arcade Library

Arcade is an easy-to-learn Python library for creating 2D games and more. The
friendly API caters to both beginners and experts alike. Do you want to craft
craft your take on a 2D classic, or explore the full power of shaders? It's up to you.

## Documentation and Community

Arcade provides documentation for every experience level.

The skill_tree or main-page-tutorials are good
for getting started. You can explore the following to
learn more:

* faq has fast answers to common questions
* intro_learning_resources covers the documentation types
* how-to-get-help if you're unsure where to start


   <br/>

## More than Open Source

Arcade makes sure licensing concerns stay out of your way:

* Arcade's code uses the permissive_mit
* The built-in resources <resources> don't require attribution
* Remixing example code <example-code> is encouraged


   <br/>

This ensures you are free to create, including commercial projects <faq_commercial>.

## Get Involved

Let us know if you see a bug, a typo, or other room for improvement.

The following community-locations are the best places
to reach the team or ask for help:

* The `Arcade Discord`_ server
* The `Arcade GitHub`_ repo


   <br/>

See how-to-contribute to learn how you can
help improve Arcade.
