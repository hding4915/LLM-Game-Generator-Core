# Better Text Drawing with Text Objects

    :width: 500px
    :align: center
    :alt: Screenshot of drawing with text objects

This example shows how to draw text using "text objects." It can be many
times faster than using :py`arcade.draw_text`.

* If you want to be as efficient as possible, see drawing_text_objects_batch.
* If you are looking for something simpler, see drawing_text.
